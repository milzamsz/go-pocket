# Getting Started

This page is a starter placeholder for `/docs/getting-started`.

## Quick Steps

1. Install Go, `templ`, Task, Tailwind, and templUI pinned versions.
2. Run `go mod download`.
3. Run `templ generate`.
4. Run `task dev`.
